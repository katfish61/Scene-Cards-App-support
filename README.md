# Scene Cards — Support

Scene Cards is a document-based app for film and TV professionals to organise, plan and track scenes.

## Getting Help

**To report a bug or ask a question — [open an issue](../../issues/new)**

You can also email: [keithtunney@me.com]

## Getting Started

1. **Launch the app** — tap New Wall to create a new Scene Cards document
2. **Import a script** — use Import Script PDF or FDX from the overflow menu (iOS) or File menu (macOS)
3. **Import a schedule** — use the Calendar mode and import a one-liner PDF
4. **Tap any scene card** to open the Inspector and add notes, images and reference files

## Frequently Asked Questions

**My script imported but some scenes are missing**
Check that the file is a screenplay PDF, not a one-liner schedule. The importer looks for INT/EXT scene headings. Password-protected PDFs are not supported.

**My schedule imported but shows no data**
The schedule importer expects a one-liner PDF (breakdown sheet). Standard script PDFs will not work here. Make sure the PDF has selectable text — scanned images cannot be read.

**Characters are missing from my scene cards after import**
Some scripts use watermarking that can interfere with character detection. Try re-importing, or add characters manually via the Characters button in the script panel.

**Where is my data stored?**
All data is stored locally inside the .scenecards document file. You can back it up, share it, or move it like any other file. There is no cloud sync.

**The app is available on iPhone/iPad — is there a Mac version?**
Yes — Scene Cards is a universal purchase. Download on Mac from the Mac App Store.

## Compatibility

- iPhone and iPad: iOS 17 or later
- Mac: macOS 14 (Sonoma) or later

## Privacy

Scene Cards does not collect any data. All your project files stay on your device.
